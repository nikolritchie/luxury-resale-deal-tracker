# luxury-resale-deal-tracker

## Overview

The Luxury Resale Deal Tracker is a tool designed to identify discounted designer fashion items and evaluate their resale value in secondary markets.

This project focuses on premium and contemporary brands such as Zimmermann, Staud, Self-Portrait, Ulla Johnson, Farm Rio, Ganni, and Cult Gaia.

The goal is to surface high-quality inventory that is significantly discounted at retail and compare it against real resale prices to identify potential value opportunities.

---

## How It Works

The system combines two key components:

### 1. Retail Price Discovery

Discounted items are sourced from major fashion retailers, including:

* Saks Fifth Avenue
* Nordstrom
* Nordstrom Rack
* Saks Off 5th

The focus is on identifying items with significant discounts (typically 70% or greater).

---

### 2. Resale Market Analysis

Each item is evaluated against real market data from:

* eBay (sold listings)

The tool analyzes:

* Historical sale prices
* Comparable listings
* Pricing consistency

This allows for estimation of a realistic resale value.

---

## Output

The system generates structured outputs including:

* Brand
* Product name
* Retail price
* Discount percentage
* Estimated resale value
* Comparable sales count
* Confidence level

This enables quick evaluation of whether an item is underpriced relative to market demand.
